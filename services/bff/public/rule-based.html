Algorithm

1

Activation History

Fetch all offers the customer has activated, enrich with category & brand from offer catalogue.

2

Build Preference Profile

Count category/brand activations → preference scores used in step 4.

3

Filter Activated Offers

Exclude offers the customer has already activated.

4

Score Each Candidate

+10× category score if category matches preference
+5× brand score if brand matches preference
+cashbackRate (e.g. 15% → +15)
+15 urgency bonus if expiring within 7 days

5

Sort & Return Top N

Sort descending by score, return top N (default 6).
